Calculadora Multifuncional en PCint

Este proyecto contiene una calculadora desarrollada para PCint con funciones extendidas, incluyendo operaciones básicas, áreas geométricas, estadística básica y generación de sucesión Fibonacci.

¿Qué hace este proyecto?

Implementa una calculadora multifuncional que puede realizar:

Operaciones básicas

Suma

Resta

Multiplicación

División

Cálculo de áreas, radios y diámetros

Figuras incluidas:

Círculo

Cuadrado

Triángulo

Trapecio

Estadística básica

Media

Mediana

Moda

Sucesión de Fibonacci

Ingresa un número inicial

Ingresa cuántos términos quieres

Muestra SOLO números pertenecientes a la serie Fibonacci real

¿Cómo está hecho? (Arquitectura del programa)

El código está dividido en módulos lógicos:

Menú principal
Controla la selección del usuario y valida entradas.

Operaciones básicas
Funciones independientes que realizan cálculos aritméticos.

Módulo geométrico
Contiene fórmulas validadas de área, diámetro y radio.

Módulo estadístico
Implementa algoritmos para ordenar, promediar y encontrar moda.

Módulo Fibonacci
Sigue la secuencia real y evita números incorrectos.

¿Para qué se hizo este proyecto?

Este proyecto sirve para:

Practicar estructuras de control en PCint

Aplicar funciones y validación de datos

Comprender conceptos básicos de estadística

Desarrollar pensamiento lógico

Crear un programa grande, modular y documentado

Dificultades encontradas y soluciones
Problemas:	                                            
PCint no soporta ciertas funciones nativas	 
Manejo de errores y entradas inválidas	      
Cálculo de moda sin arreglos automáticos	   
Evitar que Fibonacci generara valores incorrectos	
Solución:
Se ajustó la lógica para validar pertenencia real
Se construyó un algoritmo personalizado
Se agregaron verificadores de entrada antes de cada operación
Se implementaron validaciones manuales
